Création et édition d'un objet

Création

Pour créer un entité, vous devez tout d'abord sélectionner le mode «Insertion d'objets», représenté par:

Une fois le mode sélectionné, vous pouvez cliquer sur une zone du MCD pour ajouter tous les objets dont vous aurez besoin.
Nous allons maintenant voir comment éditer ces objets

Édition

L'édition des objets nécessite de passer au mode édition, représenté par:

Une fois le mode selectionné, vous pouvez double-cliquer sur les objets pour modifier leurs noms, y ajouter des commentaires, ou encore ajouter et modifier leurs propriétés. Pour ajouter une propriété, renseigner tous les champs nécéssaires (au moins le nom), puis cliquez sur «Créer».
Pour modifier une propriété, sélectionnez-la tout d'abord dans la liste, puis modifiez les champs qui vous intéressent. Une fois fait, cliquez sur «Modifier» pour appliquer les modifications.
